# Data

`results.csv` is downloaded here by `npm run fetch` (or automatically by
`npm run backtest`). It is the
[martj42/international_results](https://github.com/martj42/international_results)
dataset — every men's international football match since 1872, released under
CC0 1.0. It is not committed to this repository so that every run uses the
latest upstream copy.
